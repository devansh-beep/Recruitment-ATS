# Phase 2: Salesforce Setup & Configuration

## Job Recruitment & Hiring Management System

------------------------------------------------------------------------

## Business Hours

Business Hours were configured in Salesforce to define the
organization's official working time.

**Configuration Steps:** - Navigate to Setup → Business Hours\
- Create new Business Hours\
- Set working days (Monday--Friday)\
- Set working time (9:00 AM -- 6:00 PM)\
- Save

This ensures proper time calculations for workflows, SLAs, and
automation processes.

------------------------------------------------------------------------

## Roles

Roles were created to define the hierarchy and data visibility within
the recruitment system.

**Role Hierarchy Structure:**

CEO\
└── HR Manager\
  └── Recruiter

**Configuration Steps:** - Navigate to Setup → Roles\
- Click Set Up Roles\
- Create required roles\
- Assign users to appropriate roles

This hierarchy ensures proper record access and approval routing.

------------------------------------------------------------------------

## App Configuration

A custom Lightning App named **Recruitment ATS** was created.

**Navigation Items Added:** - Jobs\
- Candidates\
- Applications\
- Interviews\
- Reports\
- Dashboards

This app centralizes all recruitment-related objects in one workspace.

------------------------------------------------------------------------

## Object Tabs

Custom tabs were created for: - Job\

Tabs were added to the Recruitment ATS app for visibility in navigation.
